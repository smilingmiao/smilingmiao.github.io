一些基础大杂汇

Posted on 2020-04-04

作为一名程序员，总是绕不开设置环境变量，今天稍微总结一点。

从 /etc 说起，这里的 etc 不是缩写，是abbr. 等等，及其他 (et cetera) 的意思。

执行 cat /etc/paths 命令可以看到所有的 path

执行 echo $PATH 后可以看到以冒号:分割的所有 path，最后一个冒号:后面跟的是具体的命令路径，这里是调用 flutter 命令路径。

个人理解，最后一个冒号前的所有分号分开的每个 path 就像是“条条大路通罗马”里的大路，而具体 flutter 命令的路径就是“罗马”。

usr 是 Unix System Resource 的缩写

export 命令

通常我们直接在终端里面执行 export 和 source 命令后，仅在本机本次登录有效。想永远有效就直接在 .zshrc 文件里添加需要的环境变量。

比如为添加 flutter 的 国内镜像 设置环境变量

再执行一下 source ~/.zshrc

source 命令

souce 使得被执行文件的变量及函数加载进当前终端环境。
